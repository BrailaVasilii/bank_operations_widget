# Анализ банковских транзакций

## Описание

Данный проект предоставляет инструменты для анализа банковских транзакций. Он позволяет:
- Анализировать кешбэк по категориям.
- Распределять расходы по дням недели и рабочим/выходным дням.
- Считать сумму инвестиций через округление транзакций.
- Работать с API валют и акций.

## Основной функционал

1. **Загрузка транзакций** из файлов Excel.
2. **Анализ кешбэка** — расчёт возможного кешбэка по указанным категориям.
3. **Работа с API**:
    - Получение курсов валют через ExchangeRate API.
    - Получение цен акций через Yahoo Finance API.
4. **Инвестиции** — подсчёт суммы накоплений через округление транзакций.
5. **Отчёты**:
    - Расходы по категориям.
    - Расходы по дням недели.
    - Расходы по типам дней (рабочие/выходные).

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-repository-url
   cd your-project-folder
   ```

2. Установите зависимости с использованием **Poetry**:
   ```bash
   poetry install
   ```

3. Убедитесь, что у вас есть установленные зависимости для работы с Pandas и Requests.

## Использование

### Пример загрузки транзакций

Для загрузки транзакций из Excel файла используйте функцию `load_transactions`:

```python
from src.utils import load_transactions

transactions = load_transactions("data/transactions.xlsx")
print(transactions.head())
```

### Пример расчета кешбэка

Для анализа кешбэка по категориям используйте функцию `analyze_cashback_categories`:

```python
from src.services import analyze_cashback_categories
import pandas as pd

data = pd.DataFrame([
    {"Дата операции": "2023-10-01", "Сумма операции": 100, "Категория": "Еда"},
    {"Дата операции": "2023-10-03", "Сумма операции": 300, "Категория": "Транспорт"},
])

result = analyze_cashback_categories(data, year=2023, month=10)
print(result)
# Результат: {'Еда': 5.0, 'Транспорт': 15.0}
```

### Пример работы с API валют

Получение курсов валют для базовой валюты `"USD"`:

```python
from src.services import get_currency_rate

rates = get_currency_rate("USD", ["EUR", "RUB"])
print(rates)
# Результат: {'EUR': 0.85, 'RUB': 100.5}
```

### Пример формирования отчетов

Генерация отчета по расходам за последние 90 дней и категории "Еда":

```python
from src.reports import spending_by_category
import pandas as pd

data = pd.DataFrame([
    {"Дата операции": "2023-10-01", "Сумма операции": 150, "Категория": "Еда"},
    {"Дата операции": "2023-10-02", "Сумма операции": 300, "Категория": "Еда"}
])

total_spent = spending_by_category(data, category="Еда", date="2023-10-03")
print(total_spent)  # Результат: 450.0
```

## Тестирование

В проекте используются `pytest` и `unittest.mock` для модульного тестирования кода.

### Запуск тестов

Для запуска тестов выполните:

```bash
pytest
```

### Основные тестовые сценарии
- Проверка загрузки транзакций (`test_utils.py`).
- Проверка функций API и работы с кешбэком (`test_services.py`).
- Проверка генерации отчетов (`test_reports.py`).
- Проверка работы основного приложения (`test_main.py` и `test_views.py`).
